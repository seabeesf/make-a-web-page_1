1.1 Basics of the Web

WWW: a collection of HTML docs (basis of the web)
HTML: stands for Hypertext Markup Languages (glues everything together)
Hyperlinks: glue pages together, making them "weblike"

Major Pieces of the Web:
The web is made of three components: you, the internet, HTTP, servers, where "you" consists of yourself (!), a computer and a browser.

HTML:
Is made up of text content (what you see), mark ups (what it looks like) and references to other documents (images, videos, etc.).

Sample HTML Markups:

  • bold: <b>contents </b>
  • italics: <em>contents </em>
  • link: <TAG ATTR="value">contents </TAG>
  • TAG anchor: <a></a>
Sooo....to create a link: <a href="www.google.com">text</a>

Images: <img> ...and.... <img src="url" alt="text">
  • Why the "alt" text? Because...it is nice if the image does not display text indicates what the image might have been. this is nice for peeps who are blind or searches (I think). :)
  • Also, images don't have closing b/c they are void tags with no content (text).

White Space:
Break: <br> tag or void tag
Paragraph: <p>contents</p>

Inline v. Block Elements:
Why two?
  • <br> tag is an inline element (end of a line)
  • <p> tag is a block element
A sort of "box" is created around the text (sort of b/c it is sort of...invisible!).

1.2 Intro to Programming

Over time, you will see HTML code as a webpage / see a webpage as HTML.

HTML, CSS & Javascript are all languages and have different purposes w/in a webpage:

HTML - what's inside the webpage
CSS - style of the webpage
Java Script - the interactive elements
  • This section focuses on HTML (structure) and CSS (style).
  • These elements are organized into tree-like structures called DOMs.
  • Document Object Models - tree-like structures of a webpage
    (you will hear programmers refer to them as "DOMs")
Using Chrome, open "Developer Tools" and you can take a look at the structure of any page.

"Boxes, Boxes, Everywhere!"

All elements on a webpage can be drawn into boxes, which are difined using div tags:
  • <div></div>
  • ex.: If you have 4 tags, you will have 4 boxes on your page.
Use content w/in the tag to serve as place holders when you are:
  • <div>title</div>
  • <div>body</div>
  • <div>image</div>
